# 🚀 Agencia de Software Autónoma (14 Agentes)

Bienvenido a la factoría de software más avanzada. Este ecosistema de 14 agentes autónomos puede prospectar, vender, diseñar, programar, desplegar y monetizar proyectos de software por sí solo.

## 🤖 El Equipo de Élite
1.  **CEO**: El gran orquestador. Dirige todos los flujos de trabajo.
2.  **Vendedor**: Prospección en OpenStreetMap y cierre de ventas automático.
3.  **devPlan**: Estratega técnico y arquitecto de soluciones.
4.  **devDesign**: Director de arte y UX/UI.
5.  **dev**: Constructor de frontend (React/Next.js/Vite).
6.  **devBack**: Experto en backend, bases de datos y APIs.
7.  **devCopy**: El mago de las palabras y el marketing.
8.  **devSocial**: Growth hacker y automatizador de redes.
9.  **devMoney**: Integración de pagos (Stripe) y lógica de negocio.
10. **devAI**: Especialista en RAG, LLMs e integración de IA.
11. **devTest**: Auditor de calidad. Nada pasa si no tiene tests.
12. **devOps**: El arquitecto de despliegue y nube (Vercel/Cloud).
13. **devRefactor**: Limpiador maestro. Código impecable siempre.
14. **devDocs**: Responsable de la documentación técnica.

## 🛠️ Instalación y Configuración
1.  **Instalar**: Ejecuta `./install.ps1` (Windows) o `./install.sh` (Linux).
2.  **Configurar**: Edita `config_auth.json` con tus credenciales de Google OAuth.
3.  **Autenticar**: Corre `python scripts/auth_google.py --login` una sola vez.

## 📈 Flujo de Trabajo Recomendado
Pregúntale al CEO:
`opencode --agent ceo "Quiero crear un SaaS de reservas para consultorios médicos con pagos integrados"`

El CEO activará a los agentes necesarios en el orden correcto.

## 🏠 Seguridad
Todo el sistema está blindado. Las sesiones y claves están protegidas en el `.gitignore`.
